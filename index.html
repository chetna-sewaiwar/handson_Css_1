What is css

  1. Css stands for Cascading style Sheet
  2. Css describe how HTML element are to be displayed on screen , paper or in other media.
  3. Css is not neccesaryCss saves a lot of work. It can control the layout of multiple web pages all at once
  4. External stylesheets are storted in css files.

Css can be added to HTML document in 3 ways:

  • Inline - by using a style attribute inside html elements 1
  • internal - by using a style element in the head section
  • External - by using a link element to link to an external CSS file.
  • The most common way to add css, is to keep the styles in external Css files. However, In this tutorial we will use inline and internal styles,because this is easier to demonstrate, and easier for you to try it yourself.

When comparing Css class vs ID, the diffrence is that Css class applies a style to multiple elements . ID, on the other hand applies to a style to one unique element. ID is also special in that you can use a special URL to link directlyto an element and it's ude by java script.

In Css se;actor are use to target a specific element or range of element of web page. Once an element has been targeted, a style or set of styles can be applied to the element.

There is a wide range of selectors aviable. Two of the most commonly used are class and ID. Both are used to target element to which a style should be applied.
